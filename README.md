# employee-power-bi-project
# Employee Performance - Exploratory Data Analysis (EDA) & Dashboard Project

## 📋 Project Overview
This repository contains an end-to-end data analysis project focused on assessing and visualizing employee performance metrics. The goal of this project is to analyze work logs, differentiate between billable and non-billable hours, track client engagement, and evaluate resource operational efficiency across multiple corporate office locations. 

The project contains a comprehensive dataset, a Power BI development file (`.pbix`), and a finalized executive snapshot dashboard (`.pdf`) demonstrating clean visualization and performance breakdown.

---

## 📊 Business Metrics & Key Performance Indicators (KPIs)
Based on the synthesized data across all company operations, the primary organizational health metrics identified and tracked include:
* **Total Logged Hours:** 31,823 Hours
* **Total Billable Hours:** 15,712 Hours
* **Overall Utilization Rate (% Billable Hours):** 49.37%
* **Total Clients Seen:** 16,028 Clients
* **Missed Client Engagement Rate (% Clients Not Seen):** 32.98%
